# Emucoach_Cata_4.3.4_v19_Quest_Editor
GUI quest editor for Emucoach 4.3.4 Cata v19 **(Still betaish needs more testing and refining) Now semi-working but needs more testing and input**

##Requirements##

Python 3.11+

PyQt6>=6.5

PyMySQL>=1.1.0

##How to install required##

https://www.python.org/downloads/

```pip install PyQt6 PyMySQL```

##How to run##

```py app.py```
